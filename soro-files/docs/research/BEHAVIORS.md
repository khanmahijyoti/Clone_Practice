# Behaviors

- Header is sticky at top (`position: sticky; top: 0; z-index: 50`).
- Multiple text blocks animate in with opacity/translate transitions (`transition-all duration-1000 ease-out` classes).
- Hero/metrics sections use auto-playing videos and animated SVG assets.
- Horizontal chart/card rail uses translated flex wrapper (`transform: translateX(...)`).
- Mobile/date-label behavior is controlled by inline media-query style blocks in the chart section.
